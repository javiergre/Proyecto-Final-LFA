# Chomsky Classifier AI

Herramienta interactiva desarrollada en Python para:

- Clasificar gramáticas según la Jerarquía de Chomsky
- Analizar autómatas y Máquinas de Turing
- Generar diagramas con Graphviz
- Exportar análisis completos a PDF
- Aprender mediante juegos (flashcards, memoria, quiz)

Interfaz completamente en español, con soporte de Unicode y editor avanzado.
